## Prerequisites

- Node.js (v14 or higher recommended)
- SimSimi API Key (Get one from [SimSimi Workshop](https://workshop.simsimi.com/))

## Installation

1. **Clone the repository** (if you haven't already):

   ```bash
   git clone <repository-url>
   cd bot-simsimi-nodejs
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Configure the Environment**:
   Create a `.env` file in the root directory (or use the one created for you) and add your API key:
   ```env
   SIMSIMI_API_KEY=your_api_key_here
   SIMSIMI_LANG=id
   ```

## Usage

Start the bot with:

```bash
npm start
```

or

```bash
node bot.js
```
